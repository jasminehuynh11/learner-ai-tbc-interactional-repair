# Data Folder Guidance (Publication Scope)

This folder is for publication-approved, de-identified data artifacts.

## Include

- De-identified extracted text files (from dialogue processing workflow).
- JSON files containing coded repairs (de-identified and screened).

## Exclude

- Any raw file containing names, phone numbers, email addresses, or street-level addresses.
- Any direct or indirect personal identifiers not explicitly cleared by the research team.

## Pre-publication Checks

1. Run identifier screening over extracted text and JSON files.
2. Remove or mask personal information.
3. Confirm team approval before release.
4. Ensure files align with ethics approval and participant consent boundaries.

## Notes

- Raw source documents may be considered later after team discussion.
- Public release must prioritize privacy, ethics, and minimum necessary data for reproducibility.

# Data Availability Statement (Draft)

The materials in this repository support the repair-focused claims in the manuscript.

## Publicly Available

- Reproduction code and configuration (`code/`).
- Publication-approved de-identified extracted text and coded repairs JSON (`data/`).
- Frozen analysis outputs (`results/analysis/statistical_analysis_images/`).
- Human-validation schema/templates (`validation/`).

## Restricted / Controlled Access

Conversation-level source data may contain sensitive participant information and are not fully open by default.
Access to restricted materials can be considered on reasonable request to the corresponding author, subject to:

- ethics approval constraints,
- participant consent scope,
- de-identification and data use agreement requirements.

Ethics approval reference: `520251951264520` (Macquarie University Human Research Ethics Committee).

## Minimal Dataset for Interpretation

This package includes the minimal derived dataset and analysis outputs needed to interpret and verify reported findings without exposing unnecessary sensitive content.

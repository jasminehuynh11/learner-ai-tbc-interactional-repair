# Study Materials

This folder stores study instruments and reference materials that support
interpretation and reproducibility of the repair analyses.

## Included

- `Language Tasks.docx`: master task prompt list used across study weeks.

## Important Scope Note

- Files in this folder are documentation artifacts and are **not** consumed by
  the preprocessing pipeline.
- Pipeline input documents belong in `data/raw/` and are expected to be
  participant dialogue files named with the `#<student_id>. Week <week>` pattern.

## Machine-Readable Companion

For structured downstream use (analysis, auditing, appendices), see:

- `code/config/language_tasks.csv`

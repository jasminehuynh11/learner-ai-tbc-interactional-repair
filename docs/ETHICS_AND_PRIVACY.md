# Ethics and Privacy Notes

This repository is prepared for publication transparency while minimizing participant re-identification risk.

## Controls Applied

- Derived-data-first publication approach.
- Exclusion of out-of-scope datasets and analyses.
- Validation artifacts structured without direct personal identifiers.

## Before Public Release

Complete this checklist:

1. Confirm ethics approval allows public archiving of included files.
2. Confirm participant consent terms for repository sharing.
3. Remove or mask direct identifiers from any text excerpts.
4. Verify no secret keys or credentials are present.
5. Confirm manuscript wording matches final repository content.

## Sensitive Data Handling

If dialogue-level content is needed for audit:

- provide controlled-access process,
- require requestor justification and data use terms,
- log access decisions and dates.
